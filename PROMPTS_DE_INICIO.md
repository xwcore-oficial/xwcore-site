# Prompts de Inicio e Sincronismo Kiit

Use estes textos ao iniciar um novo chat ou ao colocar um chat existente no sincronismo operacional entre Kiit1 e Kiit.

## Novo chat neste computador

```text
Voce e o Kiit1. Antes de responder, leia D:\Dropbox\PROJETOS\_INSTRUCOES_CODEX.md.
Depois, para este projeto, leia AGENTS.md e PROJECT_STATE.md.
Use CHANGELOG.md apenas se precisar do historico detalhado.
Atualize PROJECT_STATE.md e CHANGELOG.md quando houver mudancas relevantes.
```

## Novo chat no outro computador

```text
Voce e o Kiit. Antes de responder, leia D:\Dropbox\PROJETOS\_INSTRUCOES_CODEX.md.
Depois, para este projeto, leia AGENTS.md e PROJECT_STATE.md.
Use CHANGELOG.md apenas se precisar do historico detalhado.
Atualize PROJECT_STATE.md e CHANGELOG.md quando houver mudancas relevantes.
```

## Ativar sincronismo em um chat existente

```text
A partir de agora, este chat deve seguir o sincronismo Kiit/Kiit1.
Leia D:\Dropbox\PROJETOS\_INSTRUCOES_CODEX.md, depois AGENTS.md e PROJECT_STATE.md deste projeto.
Use CHANGELOG.md apenas para historico detalhado.
Atualize PROJECT_STATE.md e CHANGELOG.md quando houver mudancas relevantes.
```

## Iniciar outro projeto dentro de D:\Dropbox\PROJETOS

```text
Vamos trabalhar em outro projeto dentro de D:\Dropbox\PROJETOS.
Antes de responder, leia D:\Dropbox\PROJETOS\_INSTRUCOES_CODEX.md.
Depois verifique se o projeto possui AGENTS.md, PROJECT_STATE.md e CHANGELOG.md.
Se nao possuir, proponha ou crie esses arquivos conforme a necessidade do projeto.
Execute git status --short --branch se o projeto estiver em Git.
```

## Registrar uma decisao importante

```text
Registre essa decisao nos arquivos do projeto.
Atualize PROJECT_STATE.md se afetar o estado atual ou proximas pendencias.
Atualize CHANGELOG.md com data e hora, fatos, dificuldades, solucoes e estado atual.
```

## Observacao importante

Salvar instrucoes em arquivos compartilhados nao torna essas regras automaticamente ativas em qualquer novo chat.

Para um chat novo usar essas regras, o usuario deve pedir explicitamente que Kiit ou Kiit1 leia `D:\Dropbox\PROJETOS\_INSTRUCOES_CODEX.md` e os arquivos locais do projeto.
